```
:root {
    --main-text-color: #000;
}

body {
    color: var(--main-text-color);
}
```


Переменные задаются:
- внутри селектора `:root` 
- у переменных впереди две черточки `--`
- регистрозависмые
- вызывается при помощи ф-ции `var()`

### Запись по умолчанию
10px - запись по умолчанию


```
button {
  box-shadow: var(--offset, 10px) 15px 0 0 #feafe8;
}
```
