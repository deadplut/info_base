Упорядоченная по позициям коллекция разного типа объектов.
Является [[Встроенные последовательности|последовательностью]]
Изменяемый тип данных.

- Может хранить вложенные списки(любой длины)
- не имеет граниац по кол-ву объектов
- имеет [[Списковое включение]]
- формально содержат ноль и более ссылок на объекты

### Операции
Наследует все операции последовательностей.

При вызове метода список изменется сам. А если ф-ция, то создается новый объект
```
l = [1]
b = l + [1] # создается новый объект
l.append(1) # изменеися список
```
