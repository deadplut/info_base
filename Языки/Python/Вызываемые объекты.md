Чтобы узнать вызываемый ли объект, необходимо использовать функцию callable()

[[Объект в Python|Объект]] можно сделать вызываемым определив в нем метод `__call__`
```
Class StickFinger:
	def __init(self, a):
		self.a = a

	def __call__(self):
		print(f'{a} was sticked')
		
s = StickFinger('asf')
s()
```
