Функция принимает [[Итерируемый объект|итерационные объекты]], возвращает специальный zip объект, содержащий кортежей

```
a = [1, 2, 3]
b = ['a', 'b', 'c']
с = [99, 99, 99, 99]
list(zip(a, b, c))

# [(1, 'a', 99), (2, 'b', 99), (3, 'b', 99)]
```

- при разном количестве элементов выведет с наименьшим количеством
