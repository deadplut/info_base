Типы данных [[Объект в Python]], которые даны изначально.

#### Главные встроенные типы данных:
- [[Коллекция]]
- Числа (Целые, Десятичные, комплексные и тд)
- Вызываемые([[Языки/Python/Функции|Функции]], [[Генераторы]], Классы, Методы)
- Прочие (None, [[Файлы]], Модуль, Экземпляры)
- Внутренние (Тип, Код, Фрейм)

Для проверки принадлежности объекта можно использовать `isinstance()`
###### Пример неизменяемых:
1) integer / float
2) [[Strings|string]] - строки
3) tuple (1,2,3), tuple(1,2,3) - [[Кортежи - tuple]]
4) [[Frozenset]]
5) Nonetype
6) [[Булевские значения|bool]]
7) [[Bytes - тип данных|bytes]]
###### Пример изменяемых
1) [[Списки|list]]
2) dict
3) [[Множества|set]]: set(1, 2, 3), {1, 2, 3}
4) [[bytearray - тип данных|bytearray]]
5) [[Файлы]]


Из этих данных следует, что Питон язык с <u>динамической типизацией</u>, так тип данных задается в моменте присвоения объекта переменной. 
При этом со [[Строгая типизация|строгой типизацией]].
