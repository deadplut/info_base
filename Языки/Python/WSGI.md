Шлюзовый интерфейс веб-серверов. те позволяет серверу общаться с веб-приложением путем передачи и отправки запросов.

Наиболее популярные WSGI-сереверы:
- uWSGI
- Gunicorn


Задачи:
1) управление потоками и процессами
2) может задействовать все доступные ядра процессора сервера
3) асинхронно обрабатывает несколько запросов
4) делает python приложение совместимым с другими приложениями
