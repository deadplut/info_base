
строка - упорядоченная коллекция символов для хранений текстовой и байтовой инфы.
Неизменяемый [[Языки/Python/Типы данных|тип данных]]. Являются [[Встроенные последовательности|последовательностью]].
При операциях изменений создаются новые объекты.
### Операции 
1) Создание
2) Получение элемента по индексу
3) Срез
4) Конкатенация - сложение 
5) Повтороение - умножение
   
### Методы
Для получения доступных методов необходимо вызвать  встроенную ф-цию `dir`.
У строк есть свои уникальные методы. Как `find`, `split` и тд.


На деле три типа данных:
1) str для Unicode `u'avs/012`
2) [[Bytes - тип данных|bytes]] для двоичных данных — `b'sp\x01am'`
3) [[bytearray - тип данных|bitearray]] - что-то типа массива для байтс
