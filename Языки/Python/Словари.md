Изменямый тип данных, значение которого передаются по ключу. 

- неупорядоченная коллекция произвольных данных
-  ключи должны быть [[Хэшируемость|хэшируемыми]]
- Досутп по ключу производится очень быстро из хэшируемости
- порядок ключей сохранен (до версии 3.7 порядок ключей НЕ сохранялся)


Представляют собой неупорядочную таблицу ссылок на объекты([[Хэш-таблицы]])
При поиске используют методы хэширования для быстрого извлечения данных


Не поддерживает операции [[Встроенные последовательности|последовательностей]] так как является отображением( в них нет понятия порядка)


```
D = {  
    frozenset('qwerty'): 2,  
    (3, 2): 2 ,
    True: 3
}
```
### Распаковки:
При помощи ** можно распоковать даже прям в словаре:

```
{1: 'qweqwe', **{'x': 3}}
```
